* Compile CROCO with OASIS but without XIOS

     cd models_YOURPROJECT
     source environment.sh
     cd croco-v2-1-3/exe_CPLOA_NOXIOS
     ./jobcomp

  .. code-block:: bash

     cd models_YOURPROJECT
     source environment.sh
     cd croco-v2-1-3/exe_CPLOA_XIOS
     ./jobcomp
